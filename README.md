# Portfolio
A polished portfolio website scaffold built with HTML, CSS, and JavaScript.

## Overview
This repository contains a professional single-page portfolio layout with:
- Hero section with branded introduction
- About section
- Featured projects
- Skills showcase
- Contact call-to-action

## Files
- `index.html` — main portfolio page
- `styles.css` — responsive styling and layout
- `script.js` — simple site behavior and year update
- `.gitignore` — ignores common development artifacts

## Usage
1. Open `index.html` in your browser.
2. Customize the text, projects, emails, and visuals.
3. Add your own images or expand sections as needed.

## Customize
Replace placeholder text with your personal details, change the email address in `index.html`, and update the project cards to reflect your actual work.
